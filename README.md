# PROJET Java – Gestion des étudiants

# Auteurs
- Emile Nibasumba
- Josue Mulumba Mutanda

# Objectif
Ce projet permet de gérer les étudiants, leurs cours et leurs notes.

# Fonctionnalités
- Ajouter ou modifier une note
- Supprimer un étudiant
- Afficher tous les étudiants
- Afficher les étudiants incomplets
- Calculer la moyenne
- Trier par moyenne
- Sauvegarder les données (CSV)
- Charger les données automatiquement

# Persistance
Les données sont enregistrées dans un fichier "resultats.csv".

# Exécution
1. Lancer 'Main.java'
2. Utiliser le menu
3. Les données sont sauvegardées automatiquement

# Concepts utilisés
- Programmation orientée objet (POO)
- ArrayList
- Gestion des exceptions
- Fichiers CSV
